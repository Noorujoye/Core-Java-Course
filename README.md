This repository contains my daily learning and practice while mastering Core Java.

Instead of solving isolated problems, I organized my learning topic by topic, including code examples, mini-projects,
notes, and practical implementations.

The goal of this repository is to build a strong Java foundation before moving to Spring Boot, Backend Development, and
Enterprise Java.

## 🏦 Banking Management System

One of the major projects in this repository is a console-based Banking Management System developed using Core Java and
JDBC.

### Features

- Customer Registration
- Secure Login
- Account Creation
- Money Transfer
- Transaction History
- Pagination
- Exception Handling
- Password Hashing (SHA-256)
- Layered Architecture
- DTO Pattern
- Repository Pattern
- Dependency Injection using AppFactory
- Input Validation
- Database Transactions
